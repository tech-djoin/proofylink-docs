#  Input Decision

## Decision Form (wajib diisi sebelum close check)
  - Decision dropdown:
    - APPROVED (dengan reason)
    - REJECTED (dengan reason)
    - NEED MORE INFO (request untuk subject)
    - PENDING REVIEW (ke manager)
  - Reason field (textarea untuk detail alasan)
  - Reference number (internal approval number, jika ada)
  - Authorized by (nama & email signer)
  - Timestamp otomatis
