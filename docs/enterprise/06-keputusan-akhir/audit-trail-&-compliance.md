# Audit Trail & Compliance
## Complete Audit Log
  - Who: Siapa yang melakukan action
  - What: Action apa (submit check, view result, input decision)
  - When: Exact timestamp
  - IP address & location
  - Duration (berapa lama view result)
  - Changes (audit apa yang berubah)

## Compliance Reports
  - Download audit trail (CSV untuk compliance/legal review)
  - Filter by date, user, action type
  - Digital signature / certificate of authenticity
