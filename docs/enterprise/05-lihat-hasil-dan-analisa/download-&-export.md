# Download & Export
## Export Options
  - PDF report (printable, with company branding)
  - JSON (untuk API integration)
  - CSV (untuk spreadsheet)
  - Embedded HTML (untuk embed di aplikasi)

## Report Customization
  - Select sections to include/exclude
  - Add company logo & letterhead
  - Add internal reference number
  - Add reviewer signature field (printable)
