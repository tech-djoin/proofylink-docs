# Keputusan Akhir


## Menu: Check History / Requests
  - Table display:
    - Subject name / NIK
    - Use case
    - Submitted date
    - Consent status (pending/approved/rejected)
    - Check status (pending consent / in progress / completed / failed / cancelled)
    - Action buttons (view result / resend consent / cancel)
  
  - Filter & Search:
    - By subject (nama/NIK)
    - By use case
    - By status
    - By date range
    - By consent status

## Export & Reporting
  - Download check history (CSV/Excel)
  - Filter selectable sebelum export
  - Include all metadata untuk audit trail
