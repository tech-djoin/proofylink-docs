# Tentang Token

Token adalah “saldo penggunaan” di ProofyLink. Setiap kali anda menjalankan satu proses screening/pengecekan, sistem akan menggunakan token dari saldo perusahaan anda.

## 1 token = 1 pengecekan

Di ProofyLink, **1 token setara dengan 1 kali pengecekan** (1 screening untuk 1 subject/kandidat). Ini memudahkan tim mengontrol biaya dan memprediksi kebutuhan bulanan berdasarkan volume pengecekan.

## Cek saldo token

Saldo token bisa anda lihat langsung di sidebar, pada kartu **Saldo Token**.


## Kapan token digunakan?

Token biasanya terpakai saat anda:
- Mengirim permintaan screening baru untuk kandidat/subject.
- Menjalankan pengecekan yang membutuhkan akses data untuk menghasilkan output analisa.

Kalau saldo token tidak mencukupi, anda perlu melakukan top up terlebih dahulu sebelum melakukan pengecekan berikutnya.
