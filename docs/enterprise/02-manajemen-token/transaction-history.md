# Lihat Riwayat Pengunaan

Riwayat Transaksi Token menampilkan jejak penambahan (top up) dan penggunaan token, sehingga anda bisa memantau saldo, kebutuhan pembelian token, dan melakukan audit internal bila diperlukan. Halaman ini juga membantu menelusuri transaksi tertentu melalui nomor referensi dan status transaksi.

## Kapan halaman ini digunakan?

Gunakan Riwayat Transaksi Token ketika anda ingin:
- Memastikan top up sudah tercatat dan saldo bertambah.
- Mengetahui token terpakai untuk aktivitas apa (mis. proses verifikasi/pengecekan).
- Melakukan rekonsiliasi pembayaran (mencocokkan transaksi dengan bukti pembayaran internal).

## Informasi penting yang tersedia

Di halaman ini, anda dapat melihat:
- **Saldo saat ini** dan ringkasan penggunaan token (misalnya penggunaan bulan berjalan).
- Daftar transaksi token dengan informasi utama seperti: jenis transaksi (top up / usage), tanggal, jumlah token (+/-), keterangan, nomor referensi, dan status.

## Cara membaca transaksi

Panduan cepat:
- Transaksi **top up** biasanya menambah saldo (jumlah bertanda positif).
- Transaksi **usage** mengurangi saldo (jumlah bertanda negatif) dan umumnya terkait aktivitas operasional (mis. verifikasi/pengecekan).
- **Nomor referensi** dipakai untuk pelacakan jika ada pertanyaan/komplain pembayaran atau audit.
- **Status** membantu memastikan transaksi berhasil diproses; jika status tidak sukses, token bisa jadi belum masuk/terpakai sesuai yang diharapkan.

## Tindakan yang disarankan jika ada masalah

Jika anda menemukan transaksi yang tidak sesuai:
- Cek status transaksi dan nomor referensinya, lalu cocokkan dengan bukti pembayaran atau aktivitas yang dilakukan tim.
- Jika top up belum tercermin di saldo atau ada status tidak sukses, eskalasi ke admin internal/PIC yang menangani pembayaran atau hubungi tim ProofyLink dengan menyertakan nomor referensi.
