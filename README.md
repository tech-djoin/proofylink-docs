# Cara Menjalankan Project

## 1. Install Package

Silahkan jalankan perintah berikut untuk menginstall semua dependencies:

```bash
npm install
```

## 2. Run Project

Untuk menjalankan projectnya, silahkan jalankan perintah:

```bash
npm run build
```

Setelah build selesai selanjutnya:

```bash
npm run serve
```
